# cmse492_project: Air Quality Prediction

This project explores air pollutants collected by the Belfast Centre in Northern Ireland, from January 1, 2021, to January 31, 2025. This data set includes the time recorded, air pollutants, wind speed, wind direction and temperature. The purpose of this project is to predict future nitrogen dioxide levels using time series forecasting.
